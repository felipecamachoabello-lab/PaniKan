# Panikan Full App

Esta versión incluye:
- Frontend en Next.js
- Backend API en `/api/requests`
- Conexión opcional a Supabase para base de datos
- Envío opcional de emails con Resend a Felipecamachoabello@gmail.com

Variables de entorno en Vercel:
SUPABASE_URL
SUPABASE_SERVICE_ROLE_KEY
RESEND_API_KEY
EMAIL_FROM

SQL para Supabase:
create table requests (
  id bigint generated by default as identity primary key,
  name text not null,
  phone text not null,
  service text not null,
  description text not null,
  created_at timestamptz default now()
);
